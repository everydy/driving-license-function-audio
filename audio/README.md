# Audio Folder

Place confirmed MP3 files here using the filenames in `manifest.json`.

The app is MP3-only. It does not use browser TTS.

Recommended production path:

1. Record or generate each prompt as a separate file.
2. Normalize loudness consistently.
3. Keep clips short, usually 2-5 seconds.
4. Test on iPhone Safari from a direct user tap on a playback button.

YouTube-derived audio should not be redistributed unless the usage rights are clear.
